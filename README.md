### Github

1. 安装 Git

2. 注册 GitHub 账号，验证邮箱

3. 创建文件 .gitconfig 放在用户目录下

4. 创建项目，创建文件 .gitignore，把项目上传到 GitHub

   > VCS | Import Into Version Control | Share Project on GitHub

5. 检出项目

   > VCS | Checkout from Version Control | Git | URL| Clone | Create Project

6. 提交 Ctrl + K

7. 更新 Ctrl + T
